# Warunk Abahk Website

A restaurant-themed landing page built with **HTML**, **CSS**, and **JavaScript**.

## Overview
This project presents a food and beverage brand website with navigation, product showcase, shopping cart interaction, and modal-based item details.

## Features
- Responsive navigation menu
- Product showcase sections
- Search and shopping cart interaction
- Modal-based item detail preview
- Restaurant-style visual branding

## Tech Stack
- HTML
- CSS
- JavaScript
